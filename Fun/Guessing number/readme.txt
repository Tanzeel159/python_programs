Guessing number game is a simple game in which the player must guess a number between 1 and 10 (1 and 10 included). The player has 5 chances to guess the number. If the player fails to guess the numbers, the program will reveal the guessed number. If the person guesses the number correctly, the program displays the answer. 

libraries used:
- random: random library is used to generate a random number.
